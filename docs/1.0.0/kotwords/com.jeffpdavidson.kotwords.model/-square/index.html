A square in the grid of a Crossword.

Black squares will have isBlack set to true. White squares will have a valid solution character.

Parameters

isBlack

Whether the square is black.

solution

The solution as a single character.

solutionRebus

An optional alternative solution representing a full rebus entry (multiple letters in a single square).

isCircled

Whether the square contains a circle.

entry

Optional field denoting the user's entry for the square.

entryRebus

Optional field denoting the user's rebus entry for the square.

isGiven

Whether the solution should be prefilled for the user.

number

Optional field indicating the clue number in the square. If all squares in a grid leave the number unset, then the puzzle will be assumed to use conventional crossword numbering.

foregroundColor

Optional field indicating the foreground (text) color of the square.

backgroundColor

Optional field indicating the background color of the square.

borderDirections

Optional field indicating the directions of any barred borders (walls).
